# T2 — FastAPI + RAG + ML Terminal

> **Port:** 8000
> **Role:** Machine learning inference, RAG vector search, LLM orchestration
> **Process:** `uvicorn noc_copilot:app --host 0.0.0.0 --port 8000`
> **Launcher:** `./ps13-fastapi.sh`

---

## Overview

FastAPI server that serves as the intelligence layer:
- **ML Inference**: 5 models loaded for anomaly detection, classification, and forecasting
- **RAG**: ChromaDB vector store with 7,920 indexed runbook documents
- **LLM Proxy**: Routes chat queries to Ollama with RAG context augmentation

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/health | Health check with LLM latency, RAG doc count, ML status |
| POST | /api/chat | Chat with RAG-augmented Ollama |
| POST | /predict | ML prediction from telemetry JSON |
