# T1 — Ollama LLM Terminal

> **Port:** 11434
> **Role:** Local LLM inference engine for network diagnostics
> **Process:** `ollama serve`
> **Launcher:** `./ps13-ollama.sh`

---

## Overview

Ollama runs the Qwen3-8B model with GPU acceleration via CUDA 13.3 on the RTX 4060 (8GB VRAM). It provides natural-language diagnostic responses to NOC operator queries, augmented with RAG context from ChromaDB.

## Model

- **Name**: qwen3:8b
- **Size**: 5.2 GB (Q4_K_M quantization)
- **Parameters**: 8.2 billion
- **Context window**: 32K tokens
- **Acceleration**: NVIDIA CUDA, RTX 4060

## Integration

Called by FastAPI (`noc_copilot.py`) via the `ollama` Python library:

```python
import ollama
response = ollama.chat(model="qwen3:8b", messages=[
    {"role": "system", "content": system_prompt},
    {"role": "user", "content": user_query}
])
```
